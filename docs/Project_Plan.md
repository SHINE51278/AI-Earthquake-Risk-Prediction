## Problem Statement

Earthquakes cause significant damage to life and property. Existing systems provide earthquake information, but they often lack interactive analytics and decision-support tools. This project aims to analyze historical earthquake data, identify high-risk regions, and develop an AI-powered dashboard for visualization and risk assessment.

---

## Objectives

- Analyze historical earthquake data.
- Identify earthquake-prone regions.
- Build a machine learning model for earthquake risk analysis.
- Develop an interactive dashboard.
- Support disaster preparedness through data-driven insights.
